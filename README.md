# 小宇宙播客解析器

一个简单的工具，用于解析小宇宙播客链接并获取音频下载地址。

## 功能特点

- 解析小宇宙播客链接
- 显示播客标题
- 在线播放音频
- 复制下载链接
- 美观的用户界面

## 安装

1. 克隆仓库：
```bash
git clone https://github.com/lastphantom/parsepod.git
cd parsepod
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 运行

```bash
python app.py
```

访问 `http://localhost:5000` 即可使用。

## 技术栈

- Python Flask
- TailwindCSS
- HTML5 Audio Player
